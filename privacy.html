Privacy Policy

Last updated: May 2026

Overview

Chat Margin is a browser extension that adds a local workspace next to ChatGPT conversations. It helps users keep notes, prepare prompt drafts and mark conversations for follow-up.

Local storage

Chat Margin stores notes, prompt drafts, follow-up status, UI preferences and local extension state in the user's browser. This information is used only to provide the visible extension features.

No Chat Margin server for notes

Chat Margin does not require an account and does not send notes or prompt drafts to a Chat Margin server.

Website access

The extension runs on ChatGPT pages to display the side workspace and associate local notes with the current conversation.

Support and feedback

If users choose to contact support by email, their message is sent voluntarily to hello@chatmargin.app.

Contact

For questions, contact hello@chatmargin.app.
